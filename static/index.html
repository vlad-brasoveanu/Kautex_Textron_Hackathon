    Upload CSV / Excel Sheet

    Instantly populate the active planning version by uploading an Excel (.xlsx/.xls) or CSV matrix planning sheet, or re-apply a file from your upload history below.

    Drag and drop your planning CSV or Excel file here, or click to browse

    CSV / Excel Layout Format Instructions

    Your uploaded sheet must match the standard matrix planning layout:

    • The first row must contain columns: Employee (required), Team, Location, Hours/Year (or Available hours/year (Region 100%)), Hourly Rate, followed by the specific Topic/Project columns (e.g. Agentic AI, Fuel Project). Column order doesn't matter - columns are matched by header name, so a sheet can lead with Team instead of Employee if that's how it's organized.
    • The rows below should represent employees. The intersection cells represent their allocation percentage (e.g. 20%, 0.2, or 20).
    • Grouped by team? If your sheet has a team name on its own row (e.g. a merged "Team One" cell in Excel) followed by employee rows that leave the Team/Location cell blank, that's supported too - the blank cells automatically pick up the most recent team/location above them.
    • Optional additional costs can be appended at the bottom by adding rows starting with cost names (e.g., CAD, Tooling, Prototypes, Recovery) and entering cost amounts under the corresponding topic columns.

    Uploading a different template: Every upload syncs the platform to that document - any Employee/Topic column that doesn't match a known field is automatically treated as a brand-new topic, and any employee row or topic column that isn't in the file is moved to Trash (not deleted - restore it anytime from the Trash panel in Management Panel). This means you can freely switch between different sheet templates and the dashboards, matrix, and reports will always reflect only the current document.

    Previously Uploaded Files

    Every CSV/Excel upload is kept here. Select "Apply" to re-import a previous file's data into the active planning version - useful for reverting a bad import or comparing an older sheet.
